# Insight It

## Technologies used
* ReactJS
* Bootstrap
* Reactstrap
* YUP

## To run the project
1. Download the project.
2. Open a terminal in the project folder.
3. Download the dependencies with the command: **npm install** or **yarn**
3. Run the project with the command: **yarn start**
